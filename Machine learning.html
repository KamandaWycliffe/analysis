Model Evaluation¶

Models implemented in this study will be evaluated using the classification accuracy. Essentially, Classification accuracy summarizes the performance of a classification model as the number of correct predictions divided by the total number of predictions. Therefore, the model with the highest classification accuracy better predicts whether a hashing is benign or not.

Predicting whether a hashing is benign or malign on the test data¶

In [333]:
#Define a function to return the best performing model based on the classification accuracy
def model(Model_df):
    if list(perf[0:1].Model)[0] == 'Decision Tree':
        clf = DecisionTreeClassifier(random_state=RSEED)
    elif list(perf[0:1].Model)[0] == 'Random Forest':
        clf = RandomForestClassifier(random_state=RSEED)
    elif list(perf[0:1].Model)[0] == 'Logistic Regression':
        clf = LogisticRegression(random_state=RSEED)
    elif list(perf[0:1].Model)[0] == 'Naive Bayes':
        clf = GaussianNB()
    else:
        pass
    return clf
